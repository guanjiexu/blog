Java高级工程师大纲

Posted on 2019-03-14

经历了阿里等一系列大厂的面试，从技术上来说，高级工程师这个层级在技术上不是很难。

第一知识面要广，第二用到的东西不仅要会用，并且知道其原理。除了MySQL这种大型应用系统外，自己使用过的东西，要达到，给你足够的时间，不参考其它设计案，你能够独立实现的程度。

这篇文章列举了Java高级工程师所需要的技能（不包含大数据处理），供大家参考（后面列举项时面试常考点）

  1. 计算机基础

    • 算法：排序（堆排序，快速排序，桶排序），二分查找，查找树（红黑树，B/B+树），LSM，动态规划，KMP，堆栈，LRU，限流算法，一致性Hash，Map，Leetcode中等及简单题
    • 操作系统：进程线程，虚拟内存，TCP协议（握手，拥塞控制），I/O，中断
    • 分布式：2PC/3PC，容错，Raft/Paxos，一致性Hash，负载均衡，熔断降级，调用链分析，负载均衡
    • 设计模式
    • Linux操作与常用命令

  2. Java基础

    • 集合：要深入到源码，HashMap/HashTable/ConcurrentHashMap
    • 反射
    • 泛型
    • NIO
    • 多线程：线程池，ThreadLocal，AQS，CyclicBarrir/CountDownLatch，ForkJoin，violate
    • JVM：内存区域，回收算法，类加载器（双亲委派），常见垃圾回收器

  3. MySql(Innodb)：

    • 事务：隔离级别，如何实现幻读
    • 主备，读写分离搭建
    • 分库分表
    • 索引：分布式自增，聚集索引，最左原则，Explain
    • MyBatis

  4. Redis

    • 常见数据结构及实现
    • AOF/RDB
    • Cluster和高可用搭建
    • 缓存问题的处理：击穿，预热，分布不均导致加载时间过长

  5. MongoDB（非必须）

  6. 框架

    • Spring：IOC/AOP实现，事务传播
    • SpringBoot/SpringCloud/Dobbo
    • MyBatis/Hibernate（2选1）
    • Druid
    • 全文搜索：Lucene/ElasticSearch
    • MyCat（可以学习MySQL的分布式处理方案）

  7. 中间件

    • Tomcat/nginx/haproxy：配置及调优
    • 消息队列：ActiveMQ/Kafka/RabbitMQ，消息队列作用，高可用，事务
    • Zookeeper容错

  8. 运维

    • 虚拟化：Docker/K8S
    • 自动化部署：Jenkins
    • 测试：压力测试，自动化测试，集成测试
